# The Open-Closed Game

A Prediction game

---
## Requirements

For development, you will only need Node.js and a node global package, NPM, installed in your environement.

### Node
    $ node --version
    v12.14.1

    $ npm --version
    6.14.1


## Install

    $ git clone git@github.com:doraeball22/Open-Closed_Game.git
    $ cd Open-Closed_Game
    $ npm install


## Running the project

    $ npm start

## Testing the project

    $ npm run test
